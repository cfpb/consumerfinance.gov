Lists

  • Note that modifiers are the most common use cases for this block, hence the reason there are no direct styles associated with .list or .list_item.

Link list modifier

  • The link list modifier is intended to be used for lists where each item is a link. It converts to a finger-friendly link with a large tap area on smaller screens.
