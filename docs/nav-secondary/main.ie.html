Current link modifier

Current nav link 
<a class="nav-secondary_link nav-secondary_link__current" href="#">
   Current nav link
</a>
  • .nav-secondary_link__current
  • Current links are hidden on small screens and the link button is revealed.
  • Currentl links have a left green border on large screens.

Link button modifier

<button class="nav-secondary_link nav-secondary_link__button">
    Button nav link
</button>
  • .nav-secondary_link__button
  • This modifier is only visible on smaller screens.
  • The button link modifier is used to convey the current link for small screens. It also acts as a toggle button when mixing this pattern with the expandable pattern.
  • The inline style is for demonstration purposes only and is needed because the modifier hides the element at larger screen size. Please do not use it in production.
