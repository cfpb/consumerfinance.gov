Print pages

  • Print pages are barebones printer friendly pages which are also sometimes used as the source to generate PDF's through PDF creation software.
  • The `.print` class acts a styling hook. When using the `.respond-to-print()` mixin, all stlyes will be nested under both `@media print` and `.print`.

Usage

I am a green superheader 

<div class="print">
    <h1 class="superheader">
        I am a green superheader
    </h1>
</div>
  • To trigger a print page simply add the `.print` class to a containing element.

Print header

  • Separates print header stuff from the rest of the page content.

.print-header

There should be a line underneath me =D 
<div class="print-header">
    There should be a line underneath me =D
</div>
.print-header {
  margin-bottom: 1.875em;
  padding-bottom: 1.875em;
  border-bottom: 5px solid #101820;
}
