HMDA API docs¶

A resource for data required by the Home Mortgage Disclosure Act¶

The CFPB publishes data gleaned from mortgage loan applications going back to 2007.

The API enables automated access to this rich data set. Full details on how to use it starts here.

This project is maintained by The Consumer Financial Protection Bureau.
